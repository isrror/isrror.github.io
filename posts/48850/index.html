获取不蒜子访问数据

发表于2024-06-08|更新于2024-06-09|博客
|阅读量:

前言

众所周知，不蒜子busuanzi由于注册登录功能迟迟不上线，一旦更换域名就会导致网站数据丢失，包括size_uv site_pv以及page_pv等等，本文将以博主本人域名为例，获取不蒜子的域名数据。

工具

任一能发送网络请求的工具，例如postman等等，我用的是Talend API Tester
环境：

  • Windows 10 家庭版 版本号 22H2 操作系统内部版本 19045.4412
  • Chrome v 125.0.6422.142
  • Talend API Tester v 25.13.0

操作

发送GET请求到：https://busuanzi.ibruce.info/busuanzi?jsonpCallback=BusuanziCallback
在Referer里填入自己的域名
