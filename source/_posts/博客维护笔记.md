# Twikoo评论恢复
近期发现本博客评论系统已失效，原因是Hugging Face封禁了Twikoo镜像，遂迁移至Vercel

由于Twikoo连接Atlas MongoDB数据库时，默认连接名为 **Test** 的数据库，如果需要指定数据库，则应修改为：
mongodb+srv://<db_username>:<db_password>@XXXX/{% label 数据库名称 blue %}?appName=XXXX

若Twikoo密码遗忘，则需要去MongoDB中找到config表，删除ADMIN_PASS以重置密码。

# 番组页面更新
本博客使用[hexo-bilibili-bangumi](https://github.com/HCLonely/hexo-bilibili-bangumi)作为番组插件，获取B站或者Bangumi的进度信息。
使用时，需要先在本地运行
```bash
hexo bangumi -u
```
正常来说定期更新也算方便，但是本博客主题部分直接引用外部仓库，将写作部分完全分离，迁移到Obsidian上，平时只需关注内容，此时一个外部插件就显得不那么方便了。

**在主题仓库不更改的情况下，如何在推送文章时顺便更新番剧页面呢？**

目前想法是在github deployment.yml中新增指令，可行性等待验证。
